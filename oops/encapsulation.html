Encapsulation:- encapsulation is the bundling of data and the methods that act on that data such that access to the data is restricted from outside the bundle. encapsulation allow us to control and validate the data. we can manage read and write access of objects's properties by using encapsulation. Read Only Access: if we declare getter methods only then it facilitates the read only access. write only access: if we declare setter methods only then it facilitates the write only access. Read and write access : if we declare both getter and setter methods then it facilitates both read and write access. the main advantages of using encapsulation in javascript is to provide security to the data. other benefits of encapsulation include: encapsulation guards an object against illegal access. encapsulation helps to achieve a level without revealing its complex details. this will reduce human errors. make the application more flexible and manageble. simplifies the application Conslusion: encapsulation is the tool in javascript which can be used to make the complex operation easy and manageble with making the overall application secure and easy to work
